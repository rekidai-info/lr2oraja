# LR2oraja に以下の変更を加えています

* gdx や ffmpeg など各種ライブラリを最新化
* 利用されていない Twitter 連携を削除し jar をスリム化
* OpenJDK 15 でビルド
* 起動時の最新バージョンチェック処理の実行有無の設定
* 皿チョン（beatoraja 正式導入版）
* フルスクリーン時にリフレッシュレートを指定できる機能
* 選曲画面で緑数値が変更できない不具合を修正
